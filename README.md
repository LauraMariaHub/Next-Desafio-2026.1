# 🟦 Desafio Next 2026.1 - itblue Skincare 🫧

 Este projeto faz parte do **Desafio de Next 2026.1**.  Trata-se de um sistema de e-commerce expositivo focado em produtos de skincare, utilizando tecnologias modernas de desenvolvimento web para atender aos requisitos de funcionalidade, gerenciamento de dados e responsividade.

## 🛠️ Tecnologias Obrigatórias
 Conforme as diretrizes do desafio, o projeto utiliza:
* **Next.js** (Framework)
* **TypeScript** (Tipagem estática)
* **Tailwind CSS** (Estilização)
* **Prisma** (ORM para banco de dados)

## ✨ Funcionalidades Implementadas

### Cliente
*  **Home:** Seção institucional com Missão, Visão e Valores consumidos via API (GET).
*  **Catálogo de Produtos:** Listagem dinâmica com busca funcional por nome e paginação via URL.
*  **Visualização Individual:** Detalhamento completo de cada item do banco de dados.
*  **Contato:** Formulário para envio de e-mails integrado com Resend.

### Administrativo
*  **Página de Login:** Acesso à área restrita (Interface expositiva).
*  **Dashboard de Gerenciamento:** Lista completa de produtos com Sidebar administrativa.
*  **CRUD de Produtos:** Interface para Criar, Visualizar, Editar e Excluir produtos diretamente no banco de dados.
